Regrecion Lineal 📈

¿Qué es la regresión lineal?

La regresión lineal es una técnica estadística que se utiliza para encontrar la relación entre dos variables. La regresión lineal trata de encontrar la línea que mejor se ajuste a un conjunto de puntos en un gráfico. Esta línea se llama recta de regresión y representa la tendencia de los datos.

La regresión lineal se utiliza para predecir el valor de una variable en función del valor de otra variable. Por ejemplo, se puede utilizar para predecir el salario de una persona en función de su nivel de educación, o para predecir el precio de una casa en función de su tamaño.

Para realizar un análisis de regresión lineal, se necesitan dos variables: una variable dependiente y una variable independiente. La variable dependiente es la que se quiere predecir, y la variable independiente es la que se utiliza para hacer la predicción. La recta de regresión se construye a partir de los datos de ambas variables, y se puede utilizar para hacer predicciones sobre la variable dependiente para valores de la variable independiente que no se hayan medido.

¿Como calcular la regresión lineal?

Para calcular la regresión lineal de dos variables, puedes seguir estos pasos:
  1. Escribe las coordenadas (x, y) de cada uno de los puntos que quieres incluir en el análisis de regresión lineal. La variable x será la variable independiente y la variable y será la variable dependiente.
  2. Calcula la media aritmética de cada una de las variables x e y.
  3. Calcula la diferencia entre cada punto y la media de su variable. Por ejemplo, para cada punto (x, y), calcula x - media(x) y y - media(y).
  4. Calcula el producto de las diferencias obtenidas en el paso anterior. Por ejemplo, para cada punto (x, y), calcula (x - media(x)) * (y - media(y)).
  5. Calcula el cuadrado de cada una de las diferencias obtenidas en el paso 2. Por ejemplo, para cada punto (x, y), calcula (x - media(x))^2 y (y - media(y))^2.
  6. Utiliza la siguiente fórmula para calcular el coeficiente de correlación entre x e y:
    r = sumatorio de (x - media(x)) * (y - media(y)) / sqrt(sumatorio de (x - media(x))^2 * (y - media(y))^2)
  7. Utiliza la siguiente fórmula para calcular el valor de la pendiente de la recta de regresión:
    pendiente = r * (desviación típica de y) / (desviación típica de x)
  8. Utiliza la siguiente fórmula para calcular el valor del intercepto de la recta de regresión: intercepto = media(y) - pendiente * media(x)
  9. Una vez que hayas calculado la pendiente y el intercepto, puedes escribir la ecuación de la recta de regresión de la forma y = mx + b, donde m es la pendiente y b es el intercepto.
